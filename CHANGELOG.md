## [0.1.0] - 2024-01-01

### Added

- Initial project structure and package scaffolding.
- Python source code parser using `astroid`.
- Module-level dependency graph construction with `networkx`.
- Function/method-level call graph construction.
- Git diff-based change detection.
- Impact analysis engine combining dependency and call graphs.
- Multi-factor risk scoring system.
- JSON, Markdown, and HTML report generation.
- CLI interface with `click` and `rich`.
- Git pre-commit hook integration.
- CI/CD workflows for GitHub Actions.
